# Open-Native Hub

This is the central hub repository for the Open-Native organization. It serves as the root workspace and configuration center for the organization's development environment.

## Structure

- **.opencode/**: Configuration and skills for the OpenCode agent.
- **openspec/**: Specifications and standards for the organization.
- **repos/**: Directory for storing other Open-Native repositories (ignored by git).

## Getting Started

Clone this repository to initialize your Open-Native workspace.
