# Open-Native Hub

## Purpose
This repository serves as the central hub and root workspace for the Open-Native organization. It manages the configuration, agent skills, and local development environment structure for all Open-Native projects.

## Mission
Provide a set of development toolkits and serve as a fully open-source development platform, orchestrated from this central hub.

## Structure
- **Hub Repository**: This repository (root).
- **Managed Repositories**: Other organization repositories are cloned into the `repos/` directory.

## Document status
This is a living document. It will be updated as the organization evolves and as repositories are added or changed.
